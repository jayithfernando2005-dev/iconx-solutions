# IconX Solutions — E-Commerce Mobile Website

A full-stack e-commerce mobile web application built with React, Cloudinary and Firebase.
Browse products, manage a cart, place orders, and track purchases in real time.

## Features

- User authentication with Firebase Auth
- Real-time product catalog and inventory
- Shopping cart and order management
- Admin panel for product and order control
- Attendance management system
- Trade-in calculator for devices
- Responsive mobile-first design

## Tech Stack

- React.js
- Firebase (Firestore, Auth, Hosting)
- CSS3 with mobile-first responsive design
- React Router for navigation

## Pages

- Home, Products, Cart, Order Success
- Signup, Splash Screen
- Admin Panel, Attendance Management
- Trade-In Calculator, Contact

## How to Run

1. Clone the repo
2. Run: npm install
3. Add your Firebase config to .env
4. Run: npm start
